## 📋 專題簡介

本專案是一個整合日系零售選物的柔和質感、蜜桃櫻花焦糖色系、庫存科學公式與 WMS 專題骨架的完整倉儲管理系統。

**課程定位：** 流通科技管理  
**指導教授：** 褚文明 教授  
**專題成員：** 陳玉鳳、洪依辰  
**成立時間：** 2026 年 5 月

## 🎯 核心功能

### 📊 系統架構設計
- **14 個完整頁面** - 完整的導航體系與頁面結構
- **6 個 Chart.js 動畫圖表** - 數據可視化展示
- **響應式設計（RWD）** - 支援桌面、平板、手機
- **日系零售選物視覺風格** - 蜜桃櫻花焦糖色系
- **全球搜尋功能** - 快速查找功能

### 🧮 決策支援工具
- **EOQ / EPQ 互動式計算器** - 經濟訂購量計算
- **安全庫存科學模型** - 庫存安全預警
- **庫存最佳化分析** - 數據驅動的優化建議
- **業界案例與成效分析** - 實際應用案例
- **18 個月產品迭代藍圖** - 發展規劃

### 🔧 技術棧
- **前端：** HTML + CSS + JavaScript
- **框架：** React 19 + Tailwind CSS 4
- **後端：** tRPC + Express.js
- **數據庫：** MySQL
- **庫：** Chart.js、Google Maps API、Recharts
